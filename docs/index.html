üöÄ Quick Start

Up and Running in 3 Steps

1

Clone & Install

git clone https://github.com/danilo-telnyx/telnyx-elevenlabs-bridge.git
cd telnyx-elevenlabs-bridge
python3 -m venv venv && source venv/bin/activate
pip install -r requirements.txt
2

Configure

Copy the example env and add your ElevenLabs Agent ID:

cp .env.example .env
# Edit .env with your credentials:
# ELEVENLABS_AGENT_ID=your-agent-id
# ELEVENLABS_API_KEY=key (only for private agents)
3

Run

Start the bridge, then point your Telnyx call's stream_url to it:

python bridge.py
# Listening on ws://0.0.0.0:8765/media-stream

# For development, expose with ngrok:
ngrok http 8765
